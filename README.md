# Carepatron Frontend examination

## Quick Start

1. **Clone the repository:**

    ```bash
    git clone https://github.com/shisole/carepatron_frontend.git
    ```

2. **Navigate to the project folder:**

    ```bash
    cd ui
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```
    or 
    ```bash
    yarn
    ```

5. **Run the application:**

    ```bash
    npm run dev
    ```
    ```bash
    yarn dev
    ```
